This is URL schema design document for information center.
Below is example for student.
IP address is 47.92.53.57:8080
1, retrieve list
   services/info/list/student/0/
2, create item 
   services/info/add/student/
3, view item
   services/info/student/id
4, delete item
   services/info/delete/student/id
5, update item
   services/info/update/student/id
6, delete items (post request, including ids in json format)
   services/info/delete/student/
